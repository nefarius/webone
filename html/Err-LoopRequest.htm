Seems that you have looped connection.

This may be caused by incorrect WebOne or operating system configuration.

Set UpperProxy to no in configuration file to enable WebOne direct connection to the Internet.

Make sure that only the client browser is accessing Internet via proxy, and the proxy server have a true direct connection. Note that WebOne is using system settings to access Internet. Some browsers (Internet Explorer, Safari, all Chromium-based) are using same settings as the OS where they are started.
